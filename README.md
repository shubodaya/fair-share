# Fair Share

Fair Share is a group expense tracker that helps people split costs, track
balances, and keep shared spending organized. Use it for trips, housemates, or
any shared budget where everyone wants quick clarity on who paid and who owes.

## What it does

- Create groups and invite members
- Add expenses with split types and categories
- See group totals, balances, and settlement insights
- Track activity and keep personal notes

## Use it for

- Trips and vacations
- Roommate bills
- Team budgets
- Any shared spending you want to keep transparent

## Get started

1) Create your own Firebase project for data and auth.
2) Run the app locally and connect it to your project.
3) Start a group and log your first expense.

Fair Share is built to be your shared expense hub. Set it up for your group,
invite members, and keep everything in sync.
